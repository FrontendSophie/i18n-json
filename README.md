> Translate JSON File into Other Languages

## Usage
``` bash
# install
npm install i18n-json

# generate translated json file
node index.js --src=... --lang=... --dst=...

# demo
node index.js --src=./tests/test.json --lang=en --dst=./tests/en-test.json
```
